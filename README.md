En este repertorio de Github  se implementarán y analizarán la cinemática directa, la cinemática inversa y el control cinemático de un robot diferencial. El estudiante trabajará con parámetros asignados de manera individual, desarrollará programas en MATLAB, generará resultados gráficos de calidad y producirá animaciones del movimiento del robot. También comparará el comportamiento del sistema con y sin saturación de actuadores.
